# Fitness Project

Full-stack fitness tracking application built with Node.js, Express, and MongoDB.

## Features
- Create workouts
- Get all workouts
- REST API using Express
- MongoDB integration

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## Setup

1. Clone the repo
2. Install dependencies:
   npm install
3. Create a `.env` file and add your MongoDB URI:
   MONGO_URI=your_connection_string
4. Run the server:
   npm run dev
